BELLA VISTA RESTAURANT

Project Type:
Restaurant Landing Page

Description:
Bella Vista Restaurant is a modern, responsive restaurant website built using HTML5 and CSS3. It features a clean and welcoming design with smooth hover effects and organized sections that showcase the restaurant's services and information.

Features:
- Sticky navigation bar
- Hero section with call-to-action button
- Chef's Favorites menu section
- Interactive menu cards
- Star ratings for featured dishes
- Customer reviews section
- About Us section
- Contact and Opening Hours section
- Professional footer
- Smooth scrolling navigation
- Hover animations for buttons, images, and cards
- Responsive layout

Technologies Used:
- HTML5
- CSS3
- Flexbox
- CSS Grid
